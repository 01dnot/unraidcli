# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of unraidcli
- Server management commands (info, status)
- Array management (status, start, stop)
- Docker container management (ls, ps, start, stop, restart, start-all, stop-all, stats, logs)
- VM management (ls, start, stop, restart)
- Shares management (ls, info)
- Parity check commands (status, history, start, pause, resume, cancel)
- Notifications management (ls, archive, overview)
- System metrics (CPU, memory) with watch mode
- Log viewing (ls, view, tail)
- Plugin management (ls, add, remove)
- Health check command for quick system overview
- Multi-server support with named profiles
- Multiple output formats (table, JSON, YAML)
- Watch mode for real-time monitoring
- Colorized terminal output
- Configuration management (set, show, ls, remove)
